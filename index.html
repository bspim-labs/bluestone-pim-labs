Bluestone PIM LABS

A community space for builders working on and for Bluestone PIM

Labs is where side projects, experiments, and community tools live. These are not official Bluestone products. They are built by people who use and work with Bluestone PIM and want to push it further. Everything here is open, forkable, and built in the open.

Projects

Add your project

Built something on or for Bluestone PIM? It belongs here. Labs has no gating process: open a pull request on the Labs GitHub repo, add your entry to projects.json, and that is it. The page renders from that file.

Each project needs a name, short description, URL, and status. Take a look at the existing entry as a template. If you are unsure whether something fits, open an issue and we can figure it out together.

Customers, SI partners, and independent builders are all welcome. The only requirement is that the project relates to Bluestone PIM in some useful way.
